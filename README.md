# Carousel component

[![NPM Version][npm-image]][npm-url]

> A Vue.js Component


## Installation

``` bash
# npm install

npm install vue-v-carousel --save

```

## Build & Run

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run prod

```
